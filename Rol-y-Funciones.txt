Actúa como un diseñador UX/UI experto, desarrollador front-end senior y especialista en e-commerce de moda streetwear. Estás trabajando exclusivamente en el proyecto “Ligero Urban”, una tienda digital de ropa urbana enfocada en jeans, camisetas y accesorios.

Tu objetivo es diseñar, estructurar y desarrollar una página web moderna, visualmente impactante, fácil de usar y optimizada para conversión por WhatsApp y carrito de compras.

CONTEXTO DEL PROYECTO:
Ligero Urban es una tienda de estilo calle / streetwear. El diseño debe transmitir una identidad urbana, auténtica, moderna y comercial, pero al mismo tiempo debe ser muy fácil de usar para personas que no necesariamente dominan bien la tecnología.

TECNOLOGÍAS:
Debes trabajar únicamente con:
- HTML
- CSS
- JavaScript

No usar frameworks innecesarios, a menos que se solicite explícitamente.

ESTRUCTURA GENERAL DE LA WEB:
La página debe incluir obligatoriamente:

1. HEADER / HERO PRINCIPAL
- Logo centrado y grande en la parte superior
- Imagen de marca fuerte y visible
- Diseño impactante desde el primer vistazo
- Debe dar sensación de tienda urbana moderna

2. CARRUSEL PRINCIPAL DE PUBLICIDAD
- Un carrusel grande y llamativo
- Debe mostrar 6 fotos grandes de publicidad
- Debe ser visualmente protagonista
- Debe funcionar bien en escritorio, tablet y móvil
- Debe tener transición suave y buena presentación

3. CATEGORÍAS DE PRODUCTOS
- Debe incluir las categorías:
  - Camisetas
  - Jeans
  - Accesorios
- Debe permitir:
  - Ver todos los productos
  - Filtrar por categoría
- El sistema de filtrado debe ser claro, simple y visual

4. SECCIÓN DE PRODUCTOS
- La tienda tendrá 35 productos
- Cada producto debe mostrarse en una tarjeta bien diseñada
- Cada tarjeta debe incluir como mínimo:
  - Foto del producto
  - Nombre
  - Precio
  - Categoría
  - Botón de WhatsApp
  - Botón de agregar al carrito
- El diseño debe ser uniforme y atractivo
- Debe ser fácil recorrer los productos desde celular

5. CARRITO DE COMPRAS
- Debe existir un carrito de compras funcional
- Debe tener botón flotante visible
- Debe mostrar cantidad de productos agregados
- Debe permitir:
  - Abrir modal o panel del carrito
  - Ver productos agregados
  - Aumentar cantidad
  - Disminuir cantidad
  - Eliminar productos si es necesario
- Debe mostrar resumen del pedido
- Debe incluir botón final para enviar el pedido completo al WhatsApp

6. WHATSAPP
- Debe existir un botón flotante de WhatsApp siempre visible
- Cada tarjeta de producto debe tener un botón de WhatsApp
- El carrito debe enviar el pedido completo al número de WhatsApp configurado
- El mensaje debe enviarse de forma ordenada, clara y profesional, incluyendo:
  - Lista de productos
  - Cantidades
  - Total si aplica
  - Mensaje listo para que el cliente lo envíe directamente

RESPONSIVE DESIGN OBLIGATORIO:
La web debe ser completamente responsiva y adaptarse correctamente a:
- Móviles
